# Bhiolina Bharadwaj Portfolio

Static personal website for hiring outreach and online hosting.

## Run locally

```bash
cd "/Users/pranjalnaik/Documents/App Ideas"
python3 -m http.server 5173
```

Open `http://localhost:5173`.

## Files

- `index.html`: recruiter-focused single-page website
- `styles.css`: responsive visual design
- `app.js`: reveal animation and email copy interaction
- `Bhiolina_Bharadwaj_Resume.pdf`: downloadable resume asset
